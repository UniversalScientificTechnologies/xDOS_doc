# Tools and Resources

This section provides software tools, utilities, and reference materials for working with UST radiation detectors.

- **[SD Card Preparation](sdcard_preparation)** — how to correctly format SD cards for use with AIRDOS, SPACEDOS, and GEODOS devices
- **[UST Dosimeters File Format](../xdos_format)** — description of the output data format produced by AIRDOS and LABDOS detectors
- **[DOSVIEW](../dosview)** — desktop log viewer for quick inspection and analysis of detector data
- **[DOSPORTAL](../dosportal)** — web-based portal for storing, managing, and sharing measurement data
